JQuery Effect-sliding
Metode Geser jQuery dengan jQuery Anda dapat membuat efek geser pada elemen. jQuery memiliki metode slide berikut: slideDown() slideUp() slideToggle() Metode jQuery slideDown()digunakan untuk menggeser elemen ke bawah. Metode jQuery slideUp()digunakan untuk menggeser elemen ke atas. Metode jQuery slideToggle()beralih antara metode slideDown()dan slideUp() . Jika elemen telah meluncur ke bawah, slideToggle()akan menggesernya ke atas. Jika elemen telah digeser ke atas, slideToggle()akan digeser ke bawah. Parameter kecepatan opsional dapat mengambil nilai berikut: "lambat", "cepat", milidetik. Parameter callback opsional adalah fungsi yang akan dijalankan setelah penggeseran selesai.
